# Pokemon-Web-Idle
Pokémon Web Idle é um jogo do gênero "Idle Battle RPG" projetado para rodar de forma prática e contínua diretamente no seu navegador web (baseado em Chromium).

📖 Sobre o Jogo
O projeto transforma a experiência de gerenciar e batalhar com Pokémon em um jogo ocioso (Idle RPG) que roda enquanto você utiliza o navegador. Desenvolvido majoritariamente em JavaScript (99.0%), o jogo funciona como uma extensão de navegador web.

# Pokemon-Web-Idle
Pokémon Web Idle is a "Idle Battle RPG" game designed to run practically and continuously directly in your web browser (based on Chromium).

📖 About the Game
The project transforms the experience of managing and fighting with Pokémon into an idle (Idle RPG) game that runs while you use the browser. Developed mainly in JavaScript (99.0%), the game works as a web browser extension.
